# WellNet

WellNet is a client-side Minecraft mod for Forge 1.20.1 that monitors your session health and adapts client settings to keep the game smooth and responsive in heavily modded environments.

> Target: Minecraft 1.20.1 (Forge 47.4.10)  
> Environment: Client-side only

## Features

- Lightweight core that runs health checks on your client session and connection.
- Diagnostic module that logs useful runtime information without spamming your log.
- Adaptive client settings module that can tune render distance and related options to keep performance stable.

## Installation

1. Install the matching Forge version for Minecraft 1.20.1 (Forge 47.4.10 or compatible).  
2. Download the latest WellNet release `.jar`.  
3. Put the file into your `mods` folder.  
4. Start the game and check the log for `WellNetMod` messages to confirm successful loading.
